Linux - Installable - Debian linux distro and derivatives

  • Open your preferred terminal and run the following command: 
    sudo apt update && sudo apt upgrade
sudo apt install -y curl p7zip-full
curl -s https://raw.githubusercontent.com/MangDL/MangDL/master/assets/linux-install.sh | sudo bash
  • You can now use MangDL by running the following command: 
    mangdl
