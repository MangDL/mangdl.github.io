Linux - Portable - Arch linux distro and derivatives

  • Open your preferred terminal and run the following command: 
    sudo pacman -Syyu --noconfirm curl fuse p7zip
sudo modprobe fuse
sudo groupadd fuse
sudo usermod -a -G fuse $(whoami)
curl -s https://raw.githubusercontent.com/MangDL/MangDL/master/assets/linux-dl.sh | bash
  • You can now use MangDL by running the following command: 
    ./mangdl.AppImage
